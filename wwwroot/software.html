oidc-client-js samples

At work we redesigned our security model to use OpenId Connect for authentication with Identity Server 4 as the OpenId provider. I was in charge of setting up our Angular apps to authenticate with the server and for that I used a great library called oidc-client It is maintained by the same organization as Identity Server 4 and it was very interesting figuring out how to set it up correctly (we are using the Authorization Code + PKCE OAuth flow). I wanted to dive deeper into OAuth and OpenId Connect so I started writing some sample frontend apps using oidc-client. So far I have implemented one scenario, implicit flow with Azure AD B2C. The sample apps are here.

Tuxedo Cat

Tuxedo Cat is a chess engine I wrote a few years ago. I was drawn to writing a chess engine because 1) I like chess and 2) the way chess engines work is that the UI (the board, clock etc) is decoupled from the actual engine which contains the logic, and they communicate via a standardized text protocol. That meant that I could focus on what I was interested in, which was the logical problem of finding the best moves, and simply connect to an existing GUI instead of having to worry about writing my own, which I wasn't as interested in. My engine got to the point where it was able to play a complete game at a level slightly above complete incompetence, which I was pretty proud of. It's been a while since I've built the code and there are lots of branches containing aborted improvement/refactoring attempts, but the most recent version on the master branch should still work in theory. The code is here.
